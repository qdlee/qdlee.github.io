概览

命令调色板是绑定到Ctrl+Shift+p的一个交互列表，目的是执行命令。命令调色板由命令文件供给命令条目。通常，不保证创建自己的键绑定的命令是包含进.sublime-commands文件的良好候选项。

文件格式(命令文件)

命令文件使用JSON格式并且使用.sublime-commands扩展名。

下面是Packages/Default/Default.sublime-commands中的摘录：

			[
			    { "caption": "Project: Save As", "command": "save_project_as" },
			    { "caption": "Project: Close", "command": "close_project" },
			    { "caption": "Project: Add Folder", "command": "prompt_add_folder" },

			    { "caption": "Preferences: Default File Settings", "command": "open_file", "args": {"file": "${packages}/Default/Base File.sublime-settings"} },
			    { "caption": "Preferences: User File Settings", "command": "open_file", "args": {"file": "${packages}/User/Base File.sublime-settings"} },
			    { "caption": "Preferences: Default Global Settings", "command": "open_file", "args": {"file": "${packages}/Default/Global.sublime-settings"} },
			    { "caption": "Preferences: User Global Settings", "command": "open_file", "args": {"file": "${packages}/User/Global.sublime-settings"} },
			    { "caption": "Preferences: Browse Packages", "command": "open_dir", "args": {"dir": "$packages"} }
			]

caption
在命令调色板中显示的文本。

command
被执行的命令。

args
传递给command的参数。

怎样使用命令调色板

  1. 按Ctrl+Shift+P
  2. 选择命令

命令调色板会通过上下文来过滤条目，所以无论何时你打开它，你不会总是看到每个.sublime-commands文件中的所有命令。
