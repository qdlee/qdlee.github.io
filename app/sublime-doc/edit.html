编辑

概览

Sublime Text有许多编辑特性，本主题只是抓住了皮毛而已。

列选择

列选择可以用来选择一个文件中的一块矩形区域。列选择不是通过单独模式操作，而是使用多次选择。

你可以使用添加选择选择多个文本块，或者减去选择来去除一个块。

使用鼠标

Windows

  • 右键+shift
  • 或者：中键

  • 添加到选区：Ctrl
  • 从选区减去：Alt

Linux

  • 右键+shift

  • 添加到选区：Ctrl
  • 从选区减去：Alt

OS X

  • 左键+Option
  • 或者：中键

  • 添加到选区：Command
  • 从选区减去：Command+shift

使用键盘

Windows：Ctrl + Alt + Up 和 Ctrl + Alt + Down

Linux：Alt + Shift + Up 和 Alt + Shift + Down

OS X：Cnotrol + Shift + Up 和 Cnotrol + Shift + Down

多选

多选让你能高效地对你的文本作广泛的更改。任何对多选的称赞都不过分。下面是原因：

选择一些文本，然后按Ctrl + D添加一些实例。如果你想要跳过当前实例，按Ctrl + K,Ctrl + D。

如果你选的太多，按Ctrl + U来取消对当前实例的选择。

将多选转换为行

Ctrl + L扩展选择到行的末尾。Ctrl + Shift + L拆分选择到不同的行。

你可以复制多个选择的行到一个单独的缓冲区，在那里编辑，再一次以多行选择的方式选择这些内容然后将它们粘贴到第一个缓冲区。

选择文本的其它方式

列表很长。所有可用的选择都可以在Selection菜单下找到。仅举几例：

  • 选择子字Alt + Shift +
  • 扩展选区到括号Ctrl + Shift + M
  • 扩展选区到缩进Ctrl + Shift + J
  • 扩展选区到某个范围Ctrl + Shift + Space
