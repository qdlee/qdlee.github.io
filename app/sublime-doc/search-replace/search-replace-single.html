查找和替换-单个文件

查找

为了打开查找面板缓冲区，按Ctrl + F查找面板中的一些选项和查找动作可以用键盘控制：

开关正则表达式 Alt + R
开关大小写敏感 Alt + C
开关精确匹配 Alt + W
查找下一个 Enter
查找上一个 Shift + Enter
查找全部 Alt + Enter

增量查找

增量查找面板可以由Ctrl + I调出。它和正规查找面板唯一不同的在于Enter键的行为：在增量查找中，它会选中缓冲区中的下一个匹配并且为你关闭查找面板。选择这个面板或者正规查找面板主要是偏好的问题。

替换文本

你可以使用Ctrl + H打开替换面板。

替换全部 Ctrl + Alt + Enter

小贴士

在缓冲区中查找的其它方式

Goto Anything提供了操作符#用来在当前的缓冲区中查找，查看 Goto Anything命令

在缓冲区中查找的其它键绑定

这些键绑定在查找面板隐藏的时候有效

使用最近的样式向前查找 F3
使用最近的样式向后查找 Shift + F3
使用最近的样式查找全部匹配 Alt + F3

多行查找

你可以输入一个多行查找样式。要输入换行符，在查找面板中按Ctrl + Enter。注意查找面板的大小也是可以调节的。
